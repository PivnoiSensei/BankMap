# Завдання 1: Налаштування середовища розробки та прототип карти

## Мета завдання

Підготувати робоче середовище для розробки та створити прототип інтерактивної векторної карти. Дослідити можливості роботи з картами та підготувати короткий звіт для ментора.

## Частина 2: Створення Vue проєкту

### Кроки

1. Створити новий Vue проєкт:
   ```bash
   npm create vue@latest map-admin
   ```

2. Перейти в папку проєкту та встановити залежності:
   ```bash
   cd map-admin
   npm install
   ```

3. Запустити проєкт:
   ```bash
   npm run dev
   ```

## Частина 3: Прототип векторної карти

### Вимоги

1. **Векторна карта** з використанням українського тайл-сервера:
   - URL: `https://tile.openstreetmap.org.ua`

2. **Обов'язково дослідити та реалізувати**:
   - Керування шарами карти (вмикання/вимикання)
   - Можливість вимкнення шару з назвами мікрорайонів міст
   - Додавання маркерів на карту
   - Кластеризація маркерів (якщо їх багато)
   - Popup/Tooltip для маркерів

3. **Панель налаштувань** — окремий компонент для керування відображенням карти

4. **Альтернативні бібліотеки** — самостійно знайти та порівняти бібліотеки для роботи з картами OpenStreetMap. Обрати найбільш підходящу для реалізації вимог.

---

## Звіт для ментора

Підготувати короткий звіт (Markdown) з наступним змістом:

1. **Обрана бібліотека** — яку бібліотеку обрано та чому
2. **Реалізовані можливості** — короткий опис того, що вдалося реалізувати
3. **Керування шарами** — як вирішено питання вимкнення назв мікрорайонів
4. **Проблеми та обмеження** — з чим виникли труднощі
5. **Посилання на GitHub** — репозиторій з кодом прототипу
